Project Part 10: More JS and Server Get Request

This version connects the KeiTruck Hub React app to a live Express server hosted on Render. The data for all kei truck models is now fetched dynamically from the backend.

  • Server-side code (GitHub): github.com/manan3600/server-keitruckhub
  • Render server (live API): server-keitruckhub.onrender.com/api/models
  • Client-side code (GitHub): github.com/manan3600/keitruckhub
  • Client-side website: manan3600.github.io/keitruckhub

Project Part 11: More JS and Server Post Request

This version adds a fully functional form on the Models page that lets users create new kei truck entries. The app sends a POST request to the Express server, validates input on both client and server, uploads an image file, and immediately updates the React UI with the new model card.

  • Server-side code (GitHub): github.com/manan3600/server-keitruckhub
  • Render server (live API): server-keitruckhub.onrender.com/api/models
  • Client-side code (GitHub): github.com/manan3600/keitruckhub
  • Client-side website: manan3600.github.io/keitruckhub

Project Part 12: Edit and Delete Requests

This version adds full Edit and Delete functionality for all kei truck entries. Users can now update model information or remove items directly from the Models page. All updates send PUT and DELETE requests to the Express server, validate using Joi, and instantly refresh on the client without reloading the page.

  • Server-side code (GitHub): github.com/manan3600/server-keitruckhub
  • Render server (live API): server-keitruckhub.onrender.com/api/models
  • Client-side code (GitHub): github.com/manan3600/keitruckhub
  • Client-side website: manan3600.github.io/keitruckhub/#/models

Project Part 13: Server CRUD with MongoDB

In this final version, the KeiTruck Hub server is connected to MongoDB Atlas. All model data is now stored persistently in a cloud database, including images. The React client performs full CRUD using GET, POST, PUT, and DELETE routes, all validated with Joi on the server. Adding, editing, and deleting models now updates MongoDB instantly and the UI refreshes without reloading the page.

  • Server-side code (GitHub): github.com/manan3600/server-keitruckhub
  • Render server (live API): server-keitruckhub.onrender.com/api/models
  • Client-side code (GitHub): github.com/manan3600/keitruckhub
  • Client-side website: manan3600.github.io/keitruckhub/#/models
